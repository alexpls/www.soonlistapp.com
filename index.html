Soonlist for iPhone

Get reminded when your favourite movies are coming out!

Read up on upcoming movies you're interested in and easily set up customisable reminders to notify you before they'll be released.

Features

  • Syncs back to the Soonlist server to keep release dates up to date
  • Quick link to the movie's IMDB page or website
  • View a list of all release dates by country
  • Supports 3D Touch
  • Integrates with iOS 9 Spotlight search results
